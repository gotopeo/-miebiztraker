## スクレイピング戦略

### 推奨アプローチ

1. **最新公告情報ボタンを使用**
   ```javascript
   LinkSubmit('P004','4','searchBtn');
   ```

2. **ページ遷移の待機**
   - JavaScript実行後、テーブルが表示されるまで待機
   - 待機条件: テーブルの存在確認

3. **テーブルからデータ抽出**
   - セレクタ: `table tr`
   - ヘッダー行をスキップ（1行目）
   - 各行のセル（td）を解析

4. **ページネーション対応**
   - 全76件、8ページに分割
   - 次ページボタンをクリックして全ページを取得

### 代替アプローチ（日付指定検索）

公開日（公告日）を指定して検索：
- 令和の年号を選択
- 年月日を入力
- 範囲指定で過去1ヶ月分を取得

## 実装上の注意点

1. **待機時間の調整**
   - JavaScript実行後、3-5秒の待機が必要
   - テーブルの動的読み込みを待つ

2. **セッション管理**
   - Cookie の保持が必要
   - WebDriver のセッションを維持

3. **エラーハンドリング**
   - 検索条件エラーの検出
   - ページ遷移失敗時のリトライ

4. **データ解析**
   - 案件名称に更新情報が含まれる
   - 日付形式: 「R8/1/16」（令和8年1月16日）
   - 発注機関と施行番号は改行で区切られる
