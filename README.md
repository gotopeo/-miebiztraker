# MieBid Tracker (MBT)

三重県入札情報取得システム

三重県の入札情報サイトから入札案件を自動収集し、検索・通知・管理できるWebアプリケーションシステムです。

## 主な機能

### 1. 自動スクレイピング機能
- **Selenium WebDriver**を使用した三重県入札情報サイトの自動スクレイピング
- フレームセット構造とセッション管理に対応
- 手動実行およびスケジュール実行（設定のみ実装済み）

### 2. 入札情報の検索・管理
- キーワード、発注機関、日付範囲による柔軟な検索
- ページネーション対応の一覧表示
- CSVエクスポート機能

### 3. キーワード監視・通知
- ユーザーごとのキーワード登録
- 新規案件の自動マッチング
- 有効/無効の切り替え

### 4. スケジュール設定
- 日次・週次の自動実行スケジュール設定
- 実行履歴の記録と表示

### 5. ユーザー認証
- Manus OAuth認証
- ユーザーごとのデータ管理

## データベーススキーマ

### users
ユーザー情報（Manus OAuth認証）

### biddings
入札情報（案件番号、案件名、発注機関、入札日、予定価格等）

### scrapingLogs
スクレイピング実行履歴（実行日時、ステータス、取得件数、エラー情報）

### keywordWatches
キーワード監視設定（ユーザーごとの監視キーワード）

### scheduleSettings
スケジュール設定（自動実行の設定）

## セットアップ

### 前提条件
- Node.js 22.x
- pnpm
- MySQL/TiDB データベース
- Chrome/Chromium ブラウザ（Selenium用）

### インストール

```bash
# 依存パッケージのインストール
pnpm install

# ChromeDriverのビルドスクリプトを承認
pnpm approve-builds chromedriver
pnpm rebuild chromedriver

# データベースマイグレーション
pnpm db:push
```

### 開発サーバー起動

```bash
pnpm dev
```

開発サーバーが起動し、`http://localhost:3000` でアクセスできます。

### テスト実行

```bash
pnpm test
```

## 使用方法

### 1. ログイン
システムにアクセスし、Manus OAuthでログインします。

### 2. スクレイピング実行
「スクレイピング実行」ページから手動でスクレイピングを実行できます。
- 実行ボタンをクリック
- 30秒〜1分程度で完了
- 取得件数と新規件数が表示されます

### 3. 入札情報の検索
「入札情報検索」ページで以下の条件で検索できます：
- キーワード（案件名）
- 発注機関コード
- 開始日・終了日
- ページネーション対応

### 4. CSVエクスポート
検索結果をCSV形式でダウンロードできます（最大1000件）。

### 5. キーワード監視設定
「キーワード監視」ページで監視したいキーワードを登録します。
- キーワードを入力して追加
- 有効/無効の切り替え
- 削除も可能

### 6. スケジュール設定
「スケジュール設定」ページで自動実行スケジュールを設定します。
- 毎日または毎週の実行
- 実行時刻の指定
- 週次の場合は曜日を指定

**注意**: 現在、スケジュール機能は設定のみ可能で、実際の自動実行は未実装です。

## 技術的な特徴

### Seleniumスクレイピング
三重県入札情報サイトは以下の特殊な構造を持つため、Seleniumを使用しています：

1. **フレームセット構造**: 親ページからのみアクセス可能
2. **セッション依存**: 複雑なセッション管理が必要
3. **JavaScript依存**: 動的なページ生成
4. **直接アクセス制限**: 個別URLへの直接アクセスがブロック

`server/scraper.ts`でこれらの課題に対応しています。

### tRPCによる型安全なAPI
フロントエンドとバックエンド間の通信は**tRPC**を使用し、完全な型安全性を実現しています。
- APIの定義と実装が一箇所に集約
- 型推論により自動補完が効く
- ランタイムエラーを大幅に削減

### データベース設計
Drizzle ORMを使用し、型安全なデータベース操作を実現しています。
- スキーマ定義から型が自動生成
- マイグレーションの自動管理
- インデックスによる高速検索

## 既知の制限事項

### 1. ChromeDriverのバージョン不一致
現在のサンドボックス環境では、ChromeDriverのバージョン（143）とChromeのバージョン（128）が一致していません。
本番環境では適切なバージョンを使用してください。

### 2. スケジュール自動実行未実装
スケジュール設定は保存できますが、実際の自動実行機能は未実装です。
今後のアップデートで対応予定です。

### 3. 通知機能未実装
キーワード監視の設定は可能ですが、実際の通知機能（メール、Slack等）は未実装です。
